# Ruock 1.0.0

The first release of **Ruock** — a native macOS utility that adds mechanical keyboard sound effects to every keystroke. Blazing fast, lightweight, and it lives quietly in your menu bar.

## Highlights

- ⚡ Custom low-latency audio engine built on native macOS AudioQueue APIs
- 🎧 Downloadable sound library with dozens of switch profiles (Alps, Cherry MX, Topre, Holy Panda and more)
- 🔊 Preview every soundpack in your browser before installing
- 🎵 Music awareness — auto-mutes while Spotify / Apple Music / VLC is playing
- 🖱️ Mouse and trackpad click sounds
- ⌨️ Global shortcut to toggle the app, volume & pitch variation controls
- 🧰 Dedicated CLI (`ruock-cli`) for scripts and Raycast workflows
- 🌍 Localized in English, Spanish, French, Chinese, Japanese, German, Italian, Vietnamese and Portuguese

## Install

```sh
brew install --cask ded-furby/ruock/ruock
```

Requires macOS 13.5 Ventura or later.
